# 국내외 서비스 벤치마크 — StudyT2C 고도화 아이디어

국내외 에듀테크·학습 플랫폼 사례를 바탕으로, StudyT2C에 적용할 수 있는 기능 아이디어를 정리했습니다.

---

## 1. AI 튜터·질문 답변

| 출처 | 기능 | 설명 |
|------|------|------|
| 해외 AI 튜터 | **소크라틱 질문** | 정답만 알려주지 않고 단계별 질문으로 스스로 유도 (개념 이해 중심) |
| Studyable, Notilo | **음성 대화** | STT/TTS로 말하기 연습, 발음·문법 실시간 피드백 |
| 해외 | **멀티모달** | 수식 키보드, 손글씨·이미지 인식으로 “문제 보여주며 질문” |
| Duolingo/Khan | **강도·레벨** | 스킬별 강도 미터, “깨진 스킬” 등 시각화로 복습 동기 부여 |
| 해외 | **24/7 안내 문구** | “언제든 질문하세요” 등 심리적 접근성 강조 |

---

## 2. 채점·숙제·과제

| 출처 | 기능 | 설명 |
|------|------|------|
| 해외 (Tutorela 등) | **채점 시간 50~70% 절감** | 루브릭 기반 자동 채점, 선생님은 예외·코멘트만 |
| 해외 | **학생별 문항 변형** | 같은 유형, 다른 숫자/문장으로 부정행위 방지 |
| 해외 | **LMS·성적부 연동** | Canvas, Google Classroom 등에 점수/제출 상태 자동 반영 (API) |
| 에듀허브(국내) | **PDF/HWP에서 문제 추출** | 문서 업로드 → 문항 자동 추출 → 배치 |
| 해외 | **정답률·오답 유형 집계** | 문항별/유형별 통계로 “어디가 어려운지” 한눈에 |

---

## 3. 오답노트·취약점·복습

| 출처 | 기능 | 설명 |
|------|------|------|
| 웅진스마트올(국내) | **오답 유형 분류** | “찍어서 맞힌 문제” vs “몰라서 틀린 문제” 구분 후 맞춤 복습 추천 |
| GraphRAG 연구 | **오개념·인지 오류 진단** | 지식 그래프 + AI로 단순 오답 넘어 오개념까지 진단 |
| SERO/논문 | **집단지성 해설** | 같은 문제에 여러 해설·풀이 수집 후 학생에게 맞춤 노출 |
| Anki, FSRS | **간격 반복(FSRS)** | SM-2 대신 FSRS로 복습 일정 최적화, 리뷰 수 20~30% 감소 |
| Khan/Duolingo | **마스터리 레벨** | Practiced → Mastered 단계, “마스터 챌린지”로 검증 |
| 회고 학습 | **메타인지 오답노트** | “왜 틀렸는지”만이 아니라 개념 정리·주기 복습 플로우 가이드 |

## 4. 집중·이탈·참여

| 출처 | 기능 | 설명 |
|------|------|------|

## 우선 적용 추천 (StudyT2C 기준)

1. **오답 유형 세분화** — “찍어서 맞힌 것” vs “몰라서 틀린 것” 구분 (웅진스마트올 스타일)
2. **주간/월간 학부모 리포트** — 한 장 요약 + 그래프 (홈런·AI생활기록부 스타일)
3. **선생님 조기 경고 강화** — 리스크 점수 + “지원 필요” 플래그 (PowerSchool 스타일)
4. **마스터리·진행 레벨** — 과목/개념별 Practiced → Mastered (Khan 스타일)
5. **FSRS 복습 스케줄** — next_review_at을 FSRS 기반으로 (Anki/FSRS)
6. **소크라틱 질문 모드** — AI 튜터가 단계별 질문만 하고 정답은 마지막에 (선택)
7. **LMS/성적부 API** — 외부 시스템과 점수·제출 연동 (장기)
